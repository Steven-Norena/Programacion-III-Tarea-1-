## Compilación

Para compilar el programa, utiliza un compilador de C++ con soporte para C++11 o superior (como `g++`).

Ejecuta el siguiente comando en la terminal:

```bash
g++ -std=c++11 -o tensor Tensor.cpp
```

## Ejecución

Una vez compilado, ejecuta el programa con:

```bash
./tensor
```

---

## Requisitos

* Compilador C++ (g++, clang++, etc.)
* Soporte para estándar C++11 o superior
